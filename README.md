
# React CMS

A brief description of what this project does and who it's for


## Run Locally

Clone the project

```bash
  git clone https://github.com/darkcms/cms
```

Go to the project directory

```bash
  cd cms
```

Install dependencies

```bash
  npm install
```

Create env file

```bash
  cp .env.example .env
```

Start the development server

```bash
  npm run dev
```
